# aatmiya

Aatmiya is a non-profit organisation dedicated to providing elderly citizens with group activities and health services. We aim at providing older citizens with games, meetings, and comprehensive care to enhance their quality of life.

## Website Redesign

This repository contains the redesigned website for Aatmiya Foundation, built with Next.js 16.2.7 and React 19.2.x.

### Key Features

- **Warm & Welcoming Design**: Ivory theme with saffron gold accents, creating a comfortable and approachable experience
- **Responsive Design**: Mobile-first approach with breakpoints at 768px for optimal viewing on all devices
- **Accessibility**: 13:1 color contrast, 48px minimum tap targets, respects reduced motion preferences
- **Comprehensive Services**: Health checkup camps, games & activities, community meetups, wellness sessions, counseling, and companionship
- **Modern Technology**: Next.js 16 with Turbopack, Google Fonts (Playfair Display + Inter), and best practices

### Pages

- **Home**: Hero section with photo collage, impact statistics, services overview, upcoming events, testimonials, and clear CTAs
- **About**: Our story, mission, and team information
- **Events**: Upcoming and past event galleries with timeline-style cards
- **Contact**: Contact form, direct contact information, and ways to get involved

### Design Philosophy

The website embodies the meaning of "Aatmiya" (of the soul/close to heart) through:
- Warm, human-centered design that feels like a warm embrace
- Community-focused imagery and messaging
- Accessible interfaces for elderly users
- Clear calls-to-action for volunteers, donors, and service seekers

### Technology Stack

- **Framework**: Next.js 16.2.7 (React 19.2.x)
- **Styling**: Custom CSS with CSS custom properties for theme management
- **Fonts**: Playfair Display (headings) + Inter (body) via next/font/google
- **Images**: Next.js Image optimization with polaroid-style photo collage
- **Accessibility**: WCAG 2.1 AA compliant with focus states and ARIA support

### Development

This project was developed using a subagent-driven approach with:
- 10 implementation tasks completed by specialized agents
- Spec compliance reviews after each task
- Code quality checks and testing
- Final comprehensive review

### Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to see the website in development.

### Building for Production

```bash
npm run build
npm start
```
